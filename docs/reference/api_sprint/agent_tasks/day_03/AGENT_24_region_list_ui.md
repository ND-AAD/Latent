# Agent 24: Region List Widget

**Day**: 3 | **Duration**: 3-4h | **Cost**: $4-6

## Mission
Create region management UI with list view, pin/unpin controls, and region properties display.

## Deliverables
- `app/ui/region_list_widget.py` - Region list sidebar
- `app/ui/region_properties_dialog.py` - Region details dialog
- Integration with ApplicationState

## Requirements
**Region List Widget**:
- Scrollable list of regions
- Pin checkbox per region (preserves across re-analysis)
- Color swatch showing region color
- Unity strength indicator (0-100%)
- Click to select/highlight region
- Double-click to open properties

**Properties Dialog**:
- Region ID and name (editable)
- Face count
- Unity principle (description)
- Unity strength (resonance score)
- Parametric boundary visualization
- Export to file button

## Success Criteria
- [ ] Region list displays all regions
- [ ] Pin/unpin functional
- [ ] Selection highlighting working
- [ ] Properties dialog complete
- [ ] Professional appearance
- [ ] Tests pass
